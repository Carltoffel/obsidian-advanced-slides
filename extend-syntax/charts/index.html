Charts support

Advanced Slides allows to add charts using Obsidian Charts plugin syntax or Reveal.js Charts syntax.

Obsidian Charts Example:

Currently the following features are not supported:

  • Dataview Integration
  • Chart from Table
```chart
    type: bar
    labels: [Monday,Tuesday,Wednesday,Thursday,Friday, Saturday, Sunday, "next Week", "next Month"]
    series:
      - title: Title 1
        data: [1,2,3,4,5,6,7,8,9]
      - title: Title 2
        data: [5,4,3,2,1,0,-1,-2,-3]
    ```

You can find more examples in the Obsidian Charts Dokumentation
